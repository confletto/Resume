# Service Queue Management System

A real-time line-up control system for service centers built with HTML, CSS, and JavaScript.

## Features
- Real-time queue management
- Multiple service types
- Live statistics tracking
- Customer wait time monitoring
- Service history

## Technologies
- HTML5
- CSS3 (with modern gradients and animations)
- Vanilla JavaScript (ES6+)
- 
## Usage
1. Click service type buttons to add customers to queue
2. Click "Call Next Customer" to serve next person
3. Complete or skip current customer
4. Monitor real-time statistics
